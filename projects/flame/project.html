Fire Simulation

Overview

This program is able to render pixel art fire in real time on a single CPU thread. Unlike the vast majority of fire rendering methods currently available this program does not use Navier-Stokes equations or even require the use of a fluid simulator. It instead is based on simple cellular-automata like rules allowing for a much simpler system that is also able to run in real time without the use of the GPU.

Inspiration

This was primarily inspired by a project done by Softology trying to simulate mold. Some of the simulations he had on display exhibited similar properties to a failed fluid simulation I had attempted to make earlier. That got me wondering if it would be possible to approximate fluid simulation using a similar approach. Fluid simulation is a problem that has been already solved by many people so I wanted to try a slightly different problem. Being a bit of a pyromaniac I have always wanted a way to simulate fire in real time. I got the idea from the game Little Inferno and a few online demos I had seen such as this one.
