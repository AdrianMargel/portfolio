Pixel Art Girls

Explanation

This is a set of pixel art characters I did to practice my pixel art. I had plans to also do a set of male characters at the same scale. Unfortunately for the male characters the eyes seemed too big and reducing the eyes to be one pixel smaller made the eyes way too small. In pixel art there is no such thing as making something "slightly" smaller so I was only able to do a female set.

I also gave each one a kind of "powerup" state for no other reason than because I could. I didn't make them with any specific purpose in mind but I did try to stick to the style of video game characters. I'm hoping to able to use them in a future project at some point.
